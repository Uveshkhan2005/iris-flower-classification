# 🌸 Iris Flower Classification

This project is a simple machine learning application that classifies iris flowers into three species:
- Setosa
- Versicolor
- Virginica

The classification is based on flower features like sepal length, sepal width, petal length, and petal width.

---

## 📌 Project Overview

In this project, we:
- Loaded and explored the Iris dataset
- Performed data preprocessing
- Visualized data using basic plots
- Trained a machine learning model using K-Nearest Neighbors (KNN)
- Evaluated the model performance

---

## 📊 Dataset

The dataset used is the famous **Iris dataset**, which contains:
- 150 rows (samples)
- 4 input features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 1 target column (species)
